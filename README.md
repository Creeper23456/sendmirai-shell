# SendMirai-shell
## 通过Mirai http-api实现的SSH自动报警脚本

### 用法：
脚本有两个，一个mirai.sh，可直接在shell内运行，另一个sendmirai为静默执行。
如果你的http api地址需要修改，请修改mirai.sh或sendmirai脚本底下的http_addr参数。
修改config.json并填入你发送消息的QQ号以及mirai http-api的key。
